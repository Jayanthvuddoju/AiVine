### 4. TALENT POOL — ANONYMOUS CANDIDATE PAGE (/talent)

> This is the flagship differentiator page. Vendors and recruiters browse without candidates being exposed.

**Page Header**
- Headline: *"Explore Our AI Talent Pool"*
- Sub: *"Browse pre-vetted AI professionals by role, skill set, and availability. Request a full profile to connect."*
- Note: *"All profiles are anonymized. Candidate details shared only upon mutual interest."*

**Filter Bar**
- Role (dropdown multiselect)
- Experience Level (0–2 / 2–5 / 5–10 / 10+)
- Primary Skills (tags: Python, TensorFlow, PyTorch, LangChain, SQL, AWS, Azure, etc.)
- Availability (Immediate / 2 Weeks / 1 Month)
- Work Authorization (US Citizen / Green Card / H1B / OPT / CPT)
- Location Type (Remote / Hybrid / Onsite / Relocation Open)

**Candidate Card Template**
